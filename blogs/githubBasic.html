Initialization

  • git init

    Initializes a new Git repository in the current directory.

  • git clone <URL>

    Clones a repository from a URL.

Staging and Commiting

  • git add <file>

    Stages a specific file for commit.

  • git add .

    Stages all files for commit.

  • git commit -m <message>

    Commits the staged files with a message.

  • git reset HEAD <file>

    Unstages a file from the staging area.

  • git reset --hard <commit>

    Resets the working directory to a specific commit.

Branching and Merging

  • git branch

    Lists all the branches in the repository.

  • git branch <branch>

    Creates a new branch.

  • git checkout <branch>

    Switches to a different branch.

  • git merge <branch>

    Merges a branch into the current branch.

Remote Repositories

  • git remote add <name> <URL>

    Adds a remote repository.

  • git remote rm <name>

    Removes a remote repository

  • git fetch <remote>

    Fetches changes from a remote repository.

  • git pull <remote> <branch>

    Pulls changes from a remote repository.

  • git push <remote> <branch>

    Pushes changes to a remote repository.

Status and History

  • git status

    Displays the status of the working directory.

  • git log

    Displays the commit history.

  • git diff

    Shows changes between commits, commit and working tree, etc.

  • git reflog

    Shows a reflog of recent actions

  • git blame <file>

    Shows who changed what and when in a file.

Stashing and Undoing

  • git stash

    Stashes changes in the working directory.

  • git stash list

    Lists all stashes.

  • git stash apply

    Applies a saved stash.

  • git stash pop

    Applies the most recent stash and removes it from the stash list.

  • git stash drop

    Removes a stash from the stash list.

Tagging

  • git tag

    Lists all tags.

  • git tag <tag>

    Creates a new tag.

  • git tag -a <tag> -m <message>

    Creates an annotated tag.

  • git tag -d <tag>

    Deletes a tag.

  • git push --tags

    Pushes tags to a remote repository.

Ignoring Files

  • git config core.excludes .gitignore

    Configure a .gitignore file to ignore certain files.

  • git rm --cached <file>

    Removes a file from the staging area without deleting it.

Rebasing

  • git rebase <branch>

    Rebases the current branch onto another branch.

  • git rebase -i <commit>

    Interactive rebase.

Cherry-picking

  • git cherry-pick <commit>

    Picks a specific commit from another branch and applies it to the current branch
